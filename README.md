# XTMon

XTMon is a Blazor Web App (Interactive Server) for monitoring SQL Server processing data.

## Setup

1. Update the connection string in appsettings.Development.json with your SQL login.
2. Install Tailwind and build the CSS bundle:
   - npm install
   - npm run build:css

## Run

- dotnet run

For live CSS updates during development, run `npm run watch:css` in a second terminal.
